# Programming Technology Projects

This repository contains Java programming assignments and projects developed as part of the Programming Technology course.

## Projects

### 1. JavaApplication1 - Shape Analysis Tool
A Java application for loading and analyzing geometric shapes from files. The program calculates bounding boxes and provides detailed shape analysis.

**Features:**
- Load shapes from input files (Circle, Square, Regular Triangle, Regular Hexagon)
- Calculate individual shape properties and bounding boxes
- Compute overall bounding box for all shapes
- Custom exception handling for invalid shapes

**How to run:**
```bash
java javaapplication1.Main <filename>
```

### 2. Breakthrough - Board Game
A strategic board game implementation with a graphical user interface.

**Features:**
- Interactive game board
- Two-player gameplay
- GUI-based interface using Java Swing

### 3. YogiBear - Arcade Game
A classic arcade-style game featuring Yogi Bear collecting baskets while avoiding obstacles.

**Features:**
- Level-based gameplay
- High score tracking
- Multiple game sprites (Yogi Bear, Rangers, Obstacles)
- Custom level loading system

## Technologies Used
- **Language:** Java
- **GUI Framework:** Java Swing
- **Build Tool:** Apache Ant (NetBeans)
- **Testing:** JUnit
